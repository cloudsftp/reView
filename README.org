#+title: Readme

Based on [[https://github.com/rien/reStream][reStream]]

* Dependencies

You need

- Gstreamer
- Gstreamer Plugins Base
- Gstreamer Plugins Good

** Ubuntu

#+begin_src shell
sudo apt install libgstreamer1.0-0 gstreamer1.0-plugins-base gstreamer1.0-plugins-good
#+end_src

** openSUSE

#+begin_src shell
sudo zypper install gstreamer gstreamer-plugins-base gstreamer-plugins-good
#+end_src

* Usage

** Client

Options for the client can be set as command line options or environment variables.
Command line options override the environment variables.

Values with no default value in their description have to be set (for example =--ssh-key=).

*** Command Line Options

#+begin_src
Usage: review-client [OPTIONS]

Options:
      --remarkable-ip <remarkable-ip>  IP of the reMarkable tablet (default: 10.11.99.1)
      --ssh-port <ssh-port>            SSH Port used by the reMarkable tablet (default: 22)
      --ssh-key <ssh-key>              Private SSH key file path
      --tcp-port <tcp-port>            TCP port for video stream (default: 6680)
      --dark-mode                      Dark mode - invert colors (default: false)
  -h, --help                           Print help
  -V, --version                        Print version
#+end_src

*** Environment Variables

Corresponding keys:
- =REMARKABLE_IP=
- =REMARKABLE_SSH_PORT=
- =REMARKABLE_SSH_KEY_PATH=
- =REMARKABLE_TCP_PORT=
- =REMARKABLE_DARK_MODE=

* Building

** Client

Just build it

#+begin_src shell
cargo build --release --bin review-client
#+end_src

Or with [[https://dagger.io/][dagger]]

#+begin_src shell
dagger call build-client --source . export --path review-client
#+end_src

** Server

With [[https://dagger.io/][dagger]]

#+begin_src shell
dagger call build-server --source . export --path review-server.arm.static
#+end_src
